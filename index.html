Breakfast Check-in (WebApp ใช้ได้ทั้ง iPad/Notebook)

สถานะ INH: ยังไม่ได้โหลด
ไฟล์ INH ต้องมีหัวคอลัมน์: ชื่อ-นามสกุล, ห้อง, แพคเกจ (RO/RB) หรือใช้ภาษาอังกฤษ Guest Full Name, Room, Package
* ถ้าแพคเกจไม่ใช่ RO จะถือว่าเป็น RB ทั้งหมด
✅ ถ้าเป็น RO จะเด้งเตือนให้เก็บเงิน พร้อมแสดงชื่อแขก
รายการล่าสุด (วันนี้)
วิธีใช้บน iPad: เปิดลิงก์นี้ใน Safari → Share → Add to Home Screen
วิธีใช้บน Notebook: เปิดลิงก์นี้ใน Chrome/Edge → Install app (ถ้าขึ้นไอคอนติดตั้ง)
* ข้อมูล Logs จะเก็บในเครื่องแต่ละเครื่อง (local) หากต้องการย้ายข้ามเครื่อง ให้ใช้ Export/Import Logs
